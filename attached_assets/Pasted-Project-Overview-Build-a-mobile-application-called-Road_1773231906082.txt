Project Overview
Build a mobile application called RoadWatch and a public web dashboard that creates a crowdsourced real-time snapshot of traffic conditions, dangerous driving behavior, and road safety incidents.
The system collects high-precision GPS data from passengers, drivers, and motorcycle taxi riders (boda bodas) and aggregates it into a live traffic intelligence platform.
The system must work well in cities with mixed transport ecosystems including:
private cars


taxis and minibuses


buses


trucks


boda boda motorcycles


delivery riders


pedestrians reporting incidents


Primary goals:
• improve road safety
 • detect overspeeding
 • crowdsource traffic intelligence
 • protect passengers
 • provide public road safety data

Core Concept
Every road user becomes a traffic data sensor.
When a journey begins the app captures:
GPS coordinates
 speed
 route path
 timestamp
 acceleration patterns
 braking behavior
This data feeds a real-time crowdsourced traffic engine.
